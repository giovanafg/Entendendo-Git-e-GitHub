Git Cherry Pick

O que o git cherry faz?

Ele nos ajuda a recuperar algo que acabamos apagando, algo perdido.
Então para fazer isso, eu tenho que chamar no meu terminal pelo: git reflogue, onde irá me mostrar as todas as ações que tinham sido feitas, sendo assim eu localizo qual eu quero e pego o ID desse meu commit que eu tinha “perdido” e chamo no terminal por: git cherry-pick e o número do ID Assim que esse comando for feito, pronto!
Seu commit foi restaurado com sucesso :)
